FAKE NEWS DETECTION¶

Business Question: In the today generation many companies/brands spreds false statements like they are hiring and increase there data base which lead candidate to get false information so candidate will get to know weither the information is correct or not and its companies to get awareness of getting correct information.

Steps :¶

  1. Data Exploration : Explore data for trends and analyze the text for features. Here I have used wordcount, punctuations % ratio.
  2. Data Preprocessing : remove stop words, punctuations and use stemming or lemmatize the words.
  3. Feature Engineering : Tokenize words using vectorizors. I have used TFIDFvectorizor and word2vec. Add additional features from exploratory analysis. I have added word count, punctuations % ratio.
  4. ML for classification : RandomForestClassifier, GradientBoostClassifier, CNN are used for trainig and comaprison. Test split of 10% with random state 42. Train on various hyperparameters. Tune hyper parameters to optimize accuracy and recall.

In this approach CNN has returned 96% accuracy on training and .97% accuracy on test. Random forest and GradientBoost have retured accuracy of 96% and 96% respectively.

Tried out both word2vec and tfidf on the models. TFIDF has better suited for our use case.

stopwords_en=stopwords.words('english')
ps_stemmer=nltk.PorterStemmer()

def clean_text(text):
    
    clean_text=text.lower()
    clean_text=re.sub('\W+',' ',clean_text)
    clean_text=re.sub('\s+',' ',clean_text)
    clean_text=clean_text.split(' ')
    clean_text=[word for word in clean_text if word not in stopwords_en+['']]
    clean_text=[ps_stemmer.stem(word) for word in clean_text]
    clean_text=' '.join(clean_text)
    
    return clean_text

Vectorization¶

Vectorization is the process of converting textual data (unstructured data) into numerical representations that machine learning and deep learning models can understand. Since ML models work with numbers, text data must be transformed into a numerical format. TF-IDF is an abbreviation for Term Frequency Inverse Document Frequency. This is very common algorithm to transform text into a meaningful representation of numbers which is used to fit machine algorithm for prediction.

In [18]:
def w2v_vectorize(corpus,vector_size=100,model=None):
    features = []
    if model==None:
        model=Word2Vec(corpus,vector_size=vector_size,seed=42)
    for tokens in corpus:
        zero_vector = np.zeros(model.vector_size)
        vectors = []
        for token in tokens.split(' '):
            if token in model.wv:
                try:
                    vectors.append(model.wv[token])
                except KeyError:
                    continue
        if vectors:
            vectors = np.asarray(vectors)
            avg_vec = vectors.mean(axis=0)
            features.append(avg_vec)
        else:
            features.append(zero_vector)
    return model,features

def tfidf_vectorizor(corpus,vector_size=100,model=None):
    if model ==None:
        model=TfidfVectorizer(max_features=vector_size)
    model=model.fit(corpus)
    tf_vector=model.transform(corpus)
    return model,tf_vector
